# Banking System

A full-stack banking application with separate frontend and backend.

## Project Structure

```
Banking/
├── Banking-frontend/  # React + Vite frontend application
└── backend/          # Node.js + Express backend API
```

## Frontend (Banking-frontend)

Built with:
- React
- Vite
- Tailwind CSS
- React Router

### Setup
```bash
cd Banking-frontend
npm install
npm run dev
```

## Backend

Built with:
- Node.js
- Express
- MongoDB

### Setup
```bash
cd backend
npm install
npm start
```

## Features
- User authentication (register/login)
- Account management
- Money transfers
- Transaction history
- Protected routes
